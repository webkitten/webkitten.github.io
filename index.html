Pikabu stats

Сбор данных и статистика

Pikabu stats

Серия программ для сбора, обработки и анализа данных с сайта pikabu.ru. Экстремально большие объемы: 5млн единиц информации о постах, 95млн комментариев, 1200 постов в минуту. Ограничение по количеству запросов на ip удалось обойти с помощью proxy и многопоточности.


Используемые технологии

node.js(nw.js), request-enhanced, cheerio, q, q-foreach, node-cmd, express, socket.io, nedb, chart.js, jade(pug), bootstrap3, git
